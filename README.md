# Cyber Security Toolkit

A beginner cybersecurity project built using Python.


## Features
- Password Strength Checker — Checks password strength
- SHA-256 Hash Generator — Generates secure hash
- Port Scanner — Scans open ports
- File Integrity Checker — Detects file changes
- Caesar Cipher — Encrypts and decrypts text
- Network Information Tool — Shows network details
- URL Security Checker — Detects suspicious URLs
- Password Generator — Generates strong passwords

  
## Technologies Used
- Python
- Socket Programming
- Hashlib
- Networking
- Cybersecurity Concepts

  
  ## Run Project
1. Install Python
2. Clone this repository
3. Run any tool using:
python filename.py
